# 🌡️ Temperature Converter (Java)

## Overview

A simple Java console-based application that converts temperatures between:

- Celsius
- Fahrenheit
- Kelvin

Developed as part of my Java programming practice and internship learning journey.

## Features

✅ Celsius to Fahrenheit  
✅ Celsius to Kelvin  
✅ Fahrenheit to Celsius  
✅ Fahrenheit to Kelvin  
✅ Kelvin to Celsius  
✅ Kelvin to Fahrenheit  

## Technologies Used

- Java
- Scanner Class
- Methods
- Conditional Statements

## Project Structure

```
src/
└── temperature/
    └── TemperatureConverter.java
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Temperature-Converter-Java.git
```

2. Compile

```bash
javac temperature/TemperatureConverter.java
```

3. Run

```bash
java temperature.TemperatureConverter
```
